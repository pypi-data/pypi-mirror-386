GAP Character Table Library

Overview of the GAP Character Table Library

This overview contains the following views to present information about the contents of the GAP Character Table Library (version 1.3.11).

All GAP Character Table Names by Group Order
This view lists the names of all character tables that are available in the GAP Character Table Library, sorted according to their group orders.
Simple Groups by Group Order
This view lists the nonabelian finite simple groups up to order 109 and their orders. Note that not all character tables of these groups are contained in the GAP Character Table Library.

Each character table name is linked to a page with detailed information about the character table in question. If applicable then these details pages contain links to the information about the group in question in the ATLAS of Group Representations and to the database of decomposition matrices for groups from the Atlas of Finite Groups.

The contents of such a details page can be obtained in a GAP session with BrowseCTblLibInfo or DisplayCTblLibInfo.
