Installation
============

Requirements
------------
- Python >= 3.11
- A MILP/MINLP solver (e.g., Gurobi, HiGHS, CBC)

Install from PyPI::

    pip install el1xr_opt

Alternatively, install (editable) with the recommended `el1xr_opt/` layout::

    git clone https://github.com/EL1XR-dev/el1xr_opt.git
    cd el1xr_opt
    pip install -e .[dev]
