# nGraph Schemas

This directory contains the **API schemas** for nGraphs available at the following endpoint:

```plaintext
<VideoIPath-Server>/rest/v2/data/status/network/nGraphSchemaFlat/**
```

Schemas are versioned by release, e.g., `2023.4.2_nGraphSchemaFlat.json` and `2024.1.4_nGraphSchemaFlat.json`.

Note: "headers" have been removed for simplicity.
