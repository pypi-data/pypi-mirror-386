Module par_ai_core.llm_image_utils

Utilities for handling images with Language Learning Models (LLMs).

This module provides a set of functions for encoding, decoding, and processing images to facilitate their use with Language Learning Models. It includes utilities for:

  • Base64 encoding of images
  • Image type detection
  • Converting images to base64 data URLs
  • Formatting images for chat message inputs

These utilities are particularly useful when working with LLMs that support image inputs, such as GPT-4 Vision or similar models.

Usage

from par_ai_core.llm_image_utils import image_to_base64, image_to_chat_message

Convert an image file to a base64 data URL

with open('image.jpg', 'rb') as img_file: img_bytes = img_file.read() img_b64 = image_to_base64(img_bytes, 'jpeg')

Format the image for a chat message

chat_message = image_to_chat_message(img_b64)

Note

This module supports JPEG, PNG, and GIF image formats. Attempting to use unsupported formats will raise an UnsupportedImageTypeError.

def try_get_image_type(image_path: str | Path) -> Literal["jpeg", "png", "gif"]:
    """Get image type from image path.

    Args:
        image_path: Path to the image file or data URL

    Returns:
        Image type as one of: "jpeg", "png", "gif"

    Raises:
        UnsupportedImageTypeError: If image type is not supported
    """
    if isinstance(image_path, Path):
        image_path = str(image_path)
    if image_path.startswith("data:"):
        ext = image_path.split(";")[0].split("/")[-1].lower()
    else:
        ext = image_path.split(".")[-1].lower()
    if ext in ["jpg", "jpeg"]:
        return "jpeg"
    if ext in ["png"]:
        return "png"
    if ext in ["gif"]:
        return "gif"
    raise UnsupportedImageTypeError(f"Unsupported image type: {ext}")
