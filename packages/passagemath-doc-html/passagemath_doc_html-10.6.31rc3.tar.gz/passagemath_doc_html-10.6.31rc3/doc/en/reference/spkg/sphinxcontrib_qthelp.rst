Sphinx extension which outputs QtHelp documents

Type

standard

       $ pip install sphinxcontrib_qthelp\>=1.0.3

.. tab:: conda-forge:

.. tab:: Fedora/Redhat/CentOS:

.. tab:: openSUSE:

   .. CODE-BLOCK:: bash
