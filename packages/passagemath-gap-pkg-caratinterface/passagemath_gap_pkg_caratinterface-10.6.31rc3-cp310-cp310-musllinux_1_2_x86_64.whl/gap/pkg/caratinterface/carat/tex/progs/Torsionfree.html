Program Torsionfree

Usage

Torsionfree 'file' [-t]
file: bravais_TYP containg generators for a space group

Description

Decides whether the given space group is a torsion free space group.
If so, it additionally decides whether it has trivial center.

Options

-t    : Output for TeX. Do not use!

Remarks

CAUTION: The program assumes that the translation lattice is in fact Zn. This is not checked.

See also for Order.

Examples

  1. Find the fundamental group of the Handtsche-Wendt manifold!
  2. Find all Z-classes, affine classes and torsion free space groups in a given Q-class.
