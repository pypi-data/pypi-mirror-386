Program Rein

Usage

Rein 'file' [-r] [-c]
file: matrix_TYP.

Description

Calculates the pure lattices generated by the rows (columns) of each matrix in file, ie. an integral basis of all integral points in the vectorspace spanned by the rows (columns) of each matrix in file.

Options

-r  : use the row-space (default)
-c  : use the column space
