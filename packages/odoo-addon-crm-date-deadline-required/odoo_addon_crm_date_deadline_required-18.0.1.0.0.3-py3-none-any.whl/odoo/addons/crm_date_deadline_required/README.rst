Use Cases / Context
===================

This module was developed because the field "date deadline" might be
important for some companies and some salespersons don't fill it

Usage
=====

To use this module, you need to:

1. Go to CRM.
2. Create new opportunity o choose one.
3. Update any field except the expecting closing.
4. Try to save.
5. A error will be shown.
6. Fill expecting closing.
7. Save.
